## install growroot

```shell
curl -fL -o glowroot.zip "https://github.com/glowroot/glowroot/releases/download/v0.14.4/glowroot-0.14.4-dist.zip" &&
  unzip -o glowroot.zip &&
  rm glowroot.zip

```
